Hardware Quiz

The Hardware Quiz is a multistage knowledge capacity quiz that progressively gets more difficult as it continues.
Note that there may be more than 1 correct answer per question.

  1. If you visit a web page and instead something pops up that claims that your system contains viruses or that they can make your machine run faster, it is a fraud because...
    the disk can be read by webpages
    the disk cannot be read by webpages
    the memory has been backed up by a DVD, USB memory stick, etc.
    webpages want you to download software
    none of the above
  2. Given the parts, you can assemble a desktop computer in a couple of hours with a...
    Philips screwdriver
    Philips method
    Slot-headed screwdriver
    Hammer and a box of nails
    None of the above
  3. Which of the following is true?
    To boot up quickly, get a high speed Solid State Disk
    To edit video files, get a second hard drive
    Unless you are playing video games, the CPU is almost never the choke point
    Choice 2 and 3 only
    Choices 1, 2, and 3 only
  4. Fill in the blank: Every ______ years the size of a transistor on an integrated circuit is cut in half
    2
    Three
    5
    1.5
    None of the above
  5. Programs are made out of instructions that are a string of bytes. A byte can identify the following operations:
    close, open, move, compare
    insert, delete, multiply, divide
    add, subtract, multiply, divide
    add, subtract, move, compare
    None of the above
  6. The motherboard includes a socket for:
    the CPU chip (Intel or AMD)
    iPCI/PCIe slots for adapter cards
    connectors for the USB
    choices 1-3 only
    SATA disks
  7. Which of the following is true of the hard disks?
    it has moving arms
    it rotates
    it is limited by electronical considerations
    laptops have room for a second disk
    it is possible to add disks to any computer
  8. The ethernet...
    is on the main board
    rotates
    is limited by physical considerations
    connects through a plug in card
    choice 1 or choice 4
  9. Which of the following is true of GPUs?
    are specialized processors
    they rotate
    have more power than CPUs
    stands for graphic processing unit
    have less power than CPUs
  10. Which of the following connects computers to devices that have slow to medium speed?
    USB 3.0
    USB 2.0
    printers
    e-SATA
    internal disks
